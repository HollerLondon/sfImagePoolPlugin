executeUpload ( sfWebRequest $  request)

A generic upload action for creating new images and saving them with tags, not not associating with any object.

A straight forward add to pool action. Written to allow integration with other plugins or uploading contexts which don't involve the admin gen and an sfImagePoolImage form. Handles multiple files, all with same tags. Good for WYSIWYG editor integration.

Checks for a 'tag' field name and uses this to add tags. Can be a CSV list or a single string.

Only works with POST data, doesn't aim to replace any data (PUT).
